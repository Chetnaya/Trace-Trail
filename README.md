Trace-Trail

This is a replica of the popular game Line Rider, made using Unity. The game has the same mechanics as the original game, but does not currently have any obstacles or a game loop. The purpose of this project was to learn about the mechanics of Line Rider and to gain experience with Unity.

How to Play
To play the game, simply right click the mouse to draw a line on the screen and left click to erase. Once you have drawn a line, Press on space bar to start the simulation. A car will then appear on the screen and follow the path of the line you drew. You can continue to draw additional lines to create a path for the character to follow.
